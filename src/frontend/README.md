# Xtask Frontend

A production-ready web application for completing tasks and earning money, built on the Internet Computer platform.

## Features

- **Internet Identity Authentication**: Secure, privacy-preserving authentication
- **Task Management**: Browse and complete available tasks
- **Earnings Tracking**: Track task earnings and referral rewards separately
- **Referral System**: Earn $5 for each friend who completes their first task
- **Payout Management**: Request payouts and track approval status
- **Admin Panel**: Manage tasks, users, payouts, and view platform statistics
- **Glassmorphism UI**: Modern red/white premium design with smooth animations

## Technology Stack

- **Frontend**: React 19 + TypeScript + Vite
- **Styling**: Tailwind CSS with OKLCH color system
- **UI Components**: shadcn/ui (Radix primitives)
- **State Management**: React Query (TanStack Query)
- **Routing**: TanStack Router
- **Backend**: Motoko canister on Internet Computer
- **Authentication**: Internet Identity
